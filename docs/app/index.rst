ウェブアプリ・ユーザーマニュアル
=================================

ここではウェブアプリ版BBOリートベルトの使い方を説明します。

はじめて使うとき
----------------------------
#. 管理者にグループIDを発行してもらう必要があります。
#. グループIDを使ってアカウントを作成してください。
#. 測定ファイル、装置パラメータ（PRM）ファイル、初期結晶構造モデル（CIF）ファイル、解析シークエンスを用意してください。
    - 測定ファイルは、サーバーが対応した形式である必要があります。未対応である場合はエラーが出るので管理者に依頼してください。
    - PRMファイルはウェブアプリから作成できます。
    - 結晶構造は、CIF形式でしか受け付けていません。CIF形式でもフォーマットによってはエラーが出ることがあります。そのときはMaterials Projectからダウンロードしたファイルを使うか、Vestaで一度読み込んでから再出力したファイルを使うと解決することがあります。
#. 解析シークエンスを管理者に依頼して作成してもらってください。
#. ログインして、Do refinementから必要項目を入力して、BBOリートベルトの解析を開始してください。
    - Instrument parameter file, CIF files, sequenceは事前にサーバーに登録しておく必要があります。
#. 解析のタスク処理の進捗は、Task statusページから随時確認ができます。
#. 解析が終わったら、Resultsページから結果の閲覧ができます。
