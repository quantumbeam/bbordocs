#. データ利用の目的
---------------------------------
- 我々は、BBOリートベルトサーバーを通じて収集・生成されるデータを **「研究目的」と「サービス改善目的」に限って利用します**。
- 研究目的とは、学術論文誌と学術会議における発表・公表など通常研究者の研究活動を指します。
- サービス改善目的とは、BBOリートベルトサーバーの品質管理・性能評価・障害解析・機械学習モデルの改善などを指します。
- 我々は、収集・生成したデータを営利目的での二次利用は行いません。

利用者の利用目的について

- BBOリートベルトサーバーの利用者が営利目的で本サービスを利用することは禁止しません。
- ただし、利用者の都合により、我々の研究・サービス改善目的での二次利用を妨げないものとします。

研究発表における制約
---------------------------------
我々が収集・生成したデータを研究目的で解析し、その結果を発表する場合、
**２グループ以下のデータに強く依存した解析結果の発表はしません**。

これは、もともと測定データを保有していたグループが、自力でも得られる成果を我々の研究成果として扱わないようにするためです。


収集・利用するデータの範囲
---------------------------------
我々が収集・利用するデータは以下の通りです。

- アップロードされた測定データ、装置パラメータ（RPM）ファイル。
- 本サービスで生成された解析結果。

CIFファイルは、アップロードしたグループのための解析には利用されますが、我々が二次利用することはありません。


アップロードされたデータの権利と削除
---------------------------------
- 測定データおよび装置パラメータ（PRM）ファイルの知的財産権はアップロードした利用者に帰属します。
- 我々は利用規約に定められた範囲で非独占的利用権を持ちます。
- アップロードされたデータは、再現性の確保が欠かせない研究目的で利用されるため、削除には応じられない場合があります。利用者がやむを得ない事情により削除を希望する場合は、個別の事情を確認したうえで、可能な範囲で対応します。


データ公開
---------------------------------
**データベースに蓄積されたデータは、収集日から3年経過後に、
ユーザー名・グループ名などの出所情報を削除した匿名化データとして一般公開する場合があります**。

公開の目的
""""""""""""""""""""""""
これは研究者コミュニティ全体の発展に寄与することを目的としたものです。

公開データの利用条件
""""""""""""""""""""""""
- 公開データは営利目的でも利用可能です。
- 利用にあたっては公開データ専用の規約に事前に同意する必要があります。
- 公開データについては、少数グループのデータのみを対象とした研究成果発表も許容されます。

匿名化方針
""""""""""""""""""""""""
- ユーザー名とグループ名は削除します。
- 公開データ用IDは匿名化方針に沿って付与します。

---------------------------------
- 当方は、解析結果の正確性・完全性・有用性を保証しません。解析結果の解釈・利用は利用者の責任で行ってください。
- 結果の誤用等により生じた損害について、我々は責任を負いません。
- 本サービスの利用に起因する損害については、一切の責任を負いません。
- 本サービスの中断・停止による損害については、一切の責任を負いません。
